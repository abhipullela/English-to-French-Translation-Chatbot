# English-to-French Translation Chatbot

English-to-French Translation Chatbot is a Python-based chatbot developed in Google Colab that translates English text into French using natural language processing concepts. The chatbot provides an interactive and user-friendly conversational interface for basic language translation tasks.

## Features
- English to French text translation
- Interactive chatbot interface
- NLP-based text processing
- Simple and modular implementation
- Google Colab compatible project

## Technologies Used
- Python
- Google Colab
- Natural Language Processing (NLP)

## Project Structure
- `English_to_French_Translation_Chatbot.ipynb` – Main chatbot notebook
- Supporting datasets/files (if included)

## Objective
The objective of this project is to demonstrate language translation and chatbot development using Python and NLP techniques in an interactive notebook environment.

- Support for multiple languages
